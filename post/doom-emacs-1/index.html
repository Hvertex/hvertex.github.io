doom eamcs 是一个 Emacs 的配置框架，启动速度非常高效。按键使用了*spacemacs*风格。

准备使用它，首先安装它并基本配置它

安装

1
2

git clone --depth 1 https://github.com/hlissner/doom-emacs ~/.emacs.d
~/.emacs.d/bin/doom install
  • doom 命令使用
    参数 说明
    install 安装
    sync 同步 doom emacs 配置
    env 创建和注册 envvars file
    upgrade update Doom and packages
    update update packages
    purge 删除 orphaned packages & repos,and compacts them

基本配置

安装完成后，在 ~/.doomd/ 目录下默认有三个配置文件

  • init.el - doom emacs 默认的一些 mode 可以在此文件中进行开启
  • config.el - 自定针对 mode 可以进行一些配置
  • package.el - 可是进行 package

基本命令

doom emacs 默认的 leader key 是空格见 spc

  1. SPC q l 加载最后打开的 session
  2. SPC o A 打开 org-agenda
  3. SPC f r 文件历史编辑记录
  4. SPC p p 打开项目
  5. SPC RET 调到书签 bookmark 位置
  6. SPC f p 打开个人配置
  7. SPC h d h 打开文档

更新doom emacs

1
2
3
4
5

cd ~/.emacs.d
git pull
bin/doom clean
bin/doom sync
bin/doom update

完成和了解上面的内容，就可已进行简单的文本操作了 … 【继续】

Modules

Doom 有超过130多个module组成。一个module 是一个package，configuration 和 commands 组合而成的；你可以自定义开启或者关闭。只需要配置 $DOOMDIR/init.el

只需要将相应的module的注释删除然后 执行

1

doom sync

即可生效

另外doom emacs 还为一些module提供了一些可选功能具体查看doom emacs module appendix

package 管理

安装package 编辑 $DOOMDIR/package.el

1
2
3
4
5
6

 # 安装
 (package! packageName)
 # 关闭 使用：disable 这个宏
 (package! packageName :disable t)

 (disable-package! package1 package2)

对emacs的package不是很了解，如果后面想要理解可以查看doom emacs package

配置

可以通过修改 $DOOMDIR/config.el

1
2
3

# 字体
(setq doom-font (font-spec :family "$FONTNAME" :size 16))

比如 theme 或者其他想定制的配置也可以在此修改，我主要用的不深，大部分还是默认的配置
